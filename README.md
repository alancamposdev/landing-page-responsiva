# Landing Page Responsiva - ExpoAgro 2026

Landing page desenvolvida com HTML, CSS e JavaScript puro, sem frameworks,
com foco em fundamentos de responsividade e boas práticas de front-end.

## 🎯 Sobre o projeto

Uma **landing page** é uma página única, focada em um objetivo específico de conversão. Ela é usada em campanhas de marketing para direcionar o visitante a uma ação clara, como se inscrever em um evento, comprar um produto ou preencher um formulário.

Este projeto simula a landing page de um evento fictício chamado **ExpoAgro 2026**, com o _objetivo_ de praticar:

- Estruturação semântica de HTML
- Organização de CSS com classes BEM (Block Element Modifier)
- Estruturação de conteúdo persuasivo (hero, benefícios, CTA)
- Praticar layout responsivo (mobile-first)
- Praticar animações com CSS
- Praticar manipulação do DOM com JS puro

## 🛠️ Tecnologias

- HTML5
- CSS3 (Flexbox/Grid)
- JavaScript (vanilla)
